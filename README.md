# E-Commerce Web Application (ProShop)

Full-stack e-commerce platform built with React, Node.js, and MongoDB, supporting user and admin workflows.

## Tech Stack
- Frontend: React, Redux Toolkit
- Backend: Node.js, Express
- Database: MongoDB
- Auth: JWT

## Features
- User authentication and authorization
- Product browsing and search
- Shopping cart and order placement
- Admin dashboard for managing products and orders

## Highlights
- RESTful API design with Express
- State management using Redux Toolkit and RTK Query
- Secure authentication with JWT
- Scalable architecture for frontend and backend separation

## How to Run
1. Install dependencies
   npm install

2. Run backend
   npm run server

3. Run frontend
   npm start
